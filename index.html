@font-face

为了实现网页的特殊字体效果，我们可以利用css3的@font-face属性

Browser Support


* Edge and IE: The font format only works when set to be “installable”.
* Firefox: Disabled by default, but can be enabled (need to set a flag to “true” to use WOFF2).
